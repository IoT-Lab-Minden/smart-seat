◆ displayCardPresence()

void LCD::displayCardPresence ( bool  isPresent,
CardState  state 
)

The function displays the 'RFID card is present'-sign on the LCD-screen.

The color of the sign represents the current card state.

green - identified

red - unknown

blue - empty

The sign is deleted (overpainted) if no card is present.

Parameters
isPresent- a bool value, true if RFID card shall be displayed as present
state- the CardState (empty, identified, unknown)
Returns
nothing

◆ displayVoltage()

void LCD::displayVoltage ( uint16_t  voltage)

The function displays a voltage value [0-4095] on the display.

This function is mainly intended for service and setup purposes.

If you really need the function, change the variable 'serviceMode' to 'true' inside the StateMachine.

Parameters
